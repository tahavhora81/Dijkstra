User's Guide for Dijstra-Your Learning Graph!

1. Enter the topic you want to explore in the text box and click Go!

You can search for any topic that you want to start learning. The topic may range from science to linguistics, skill development and much more...

If you have any preferences then you can also mention them with your topic name.

For example:

  • "Data Science"
  • "Organic Chemistry for JEE examination"
  • "AutoCAD for skill development"

2. Select the prerequisites that are known to you and click Next!

The LLM will create a tailored learning path for you based on the prerequisites known to you.

Select at least one prerequisite for optimized results.

3. Here is the tailored learning path based on your preferences!

The learning path consists of subtopics ordered in the learning hierarchy

It also contains a short description of the topic and the name of resources you can use to learn that subtopic.

The resources range from books to online courses as well as youtube videos.

Further if you want to analyse the topic in detail, click on the Create Concept Tree button.

Also, simply click on 'Print Learning Path' button to save your learning path.

4. Whoa! There you see the concept tree is generated

The LLM creates a concept tree which is a hierarchical representation of the topics.

The parent topic expands into it's children subtopics.

You can click on individual nodes to expand into it's subtopics!

Also, simply click on 'Print Concept Tree' button to save your concept tree.
